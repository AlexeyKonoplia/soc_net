# Центральная африка

Работа заключалась в поиске географических и экономических данных по странам Центральной Африки  
Страный Центральной Африки:

1. Ангола
2. Бурунди
3. Камерун
4. Центральноафриканская Республика
5. Чад
6. Демократическая Республика Конго
7. Республика Конго
8. Габон
9. Экваториальная Гвинея
10. Сан-Томе и Принсипи


Данные находятся [тут](https://drive.google.com/drive/folders/1hfV35Bs7uULt_gMAvzJfG5wjZkD_OFBk?usp=drive_link)  
Дисциплина была посвещена социальным отношениям, теории графов и теории игр, поэтому особое внимание уделялось географическим данным для дальнейшей симуляции "игр", а экономические данные использовались преимущественно для оценки городов и составления рейтинга пригодности вершины графа (города).

## Предисловие

Блокнот никак не обработан и взят из одной из первых рабочих веток проекта. Весь курс проходился командой из 11 человек, я был ответственен за первичный сбор данных и их какую-никакую визуализацию.
